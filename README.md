# PokeDexApi

### Description

PokeDex Api is an application that uses the knowledge acquired during the second module of DevStock Academy.

---

### Instalation by Node Package Manager

```instal
git clone https://github.com/micgol512/PokeDexApi.git
cd PokeDexApi
npm install
```

---

### Start

```run
npm run dev
```
